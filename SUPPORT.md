# Support

## Getting Help

- Open a GitHub issue for bugs and technical questions.
- Include logs, config snippets (sanitized), and exact commands where possible.

## What to Include

- Quaid version
- Host/runtime (OpenClaw, MCP client, standalone CLI)
- OS and environment details
- Steps to reproduce
- Expected vs actual behavior

## Known Reality (Alpha)

Quaid is in early alpha. OpenClaw is the most mature path. Other host integrations may be experimental and less validated.
