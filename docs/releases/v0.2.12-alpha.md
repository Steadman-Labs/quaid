# Quaid v0.2.12-alpha

## Highlights

- `sqlite-vec` is now a hard installer requirement.
- Preflight now verifies `sqlite-vec` import and attempts installation automatically when missing.
- Installer now fails fast if `sqlite-vec` remains unavailable after install attempts.
- Removed optional/skip behavior for `sqlite-vec` so vector retrieval does not silently degrade.

## Notes

- If installation fails on this step, run:
  - `python3 -m pip install --user sqlite-vec`
- Then re-run installer.
