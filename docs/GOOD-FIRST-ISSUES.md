# Good First Issues

These are intentionally small, scoped tasks for new contributors.

1. Add startup warning when host auth profile is missing
   - Area: OpenClaw adapter
   - Done when: missing-profile path prints a clear actionable message.

1. Improve E2E matrix skip reason labels
   - Area: test infra
   - Done when: skipped lanes always include machine-readable reason codes.

1. Add docs example for explicit provider split (`deep` vs `fast`)
   - Area: docs
   - Done when: one complete config example is added and linked.

1. Add unit tests for no-fallback credential policy
   - Area: adapter tests
   - Done when: tests fail if env/.env fallback is reintroduced.

1. Add concise “cost safety” section to benchmark docs
   - Area: benchmark docs
   - Done when: docs explain free/subscription vs paid/API paths clearly.

1. Tighten CLI config help text for provider settings
   - Area: config CLI
   - Done when: provider option descriptions are explicit about cost impact.
